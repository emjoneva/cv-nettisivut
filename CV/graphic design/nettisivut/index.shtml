MY OWN WEBPAGE

Aloitin tekemään nettisivujani marraskuun puolen välin tienoilla, tavoitteenani havainnollistaa osaamistani front-end -puolella. Olen tehnyt sivut kokonaan itse, mikä on ollut todella opettava kokemus.

Uusia juttuja, joita opin tehdessäni omaa nettisivuani
- SSI:n hyödyntäminen .include-tiedostojen avulla
- Flexboxin asettelun kikkoja
- Yksinkertaista Javascriptiä responsiivisen navigoinnin tekemiseen

Kun kiireet kouluhommista väistyvät joululoman alkaessa, aion perehtyä vielä enemmän front-end -kehitykseen. Käyn tällä hetkellä Udemyssa Fullstack Web Dev -kurssia, jolla perehdytään sivun ulkoasun lisäksi siihen, mitä "taustalla" tapahtuu. Aion etenkin keskittyä oppimaan Javascriptiä ja parantamaan Python-taitojani.

I started creating my own webpage in November. I wanted to visualize my own skills in the world of front-end development. I've made my webpage from scratch, so I've learned a lot during the project.

New things that I've learned while making my own webpage
- making use of SSI and .include-files
- new tricks for formatting Flexbox
- simple Javascript to create a responsive navigation bar
